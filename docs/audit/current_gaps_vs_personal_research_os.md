# Current Gaps vs Personal Research OS

Implemented foundations:

- document/source/chunk/citation data model
- retrieval audit logging
- Markdown-first output generation
- Gemma4 adapter with scaffold fallback
- processing status dashboard data
- maintenance health report
- PDF source open/page jump adapter

Remaining gaps:

- Mozilla PDF.js is documented but not vendored yet.
- Highlight overlay is not implemented.
- OCR status is schema-level only.
- Graph visualization is not implemented.
- Local Gemma4 endpoint must be configured externally.
- Native PPTX is intentionally out of scope.
