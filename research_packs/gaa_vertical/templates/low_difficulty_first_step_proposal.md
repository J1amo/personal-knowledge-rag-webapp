# Low-Difficulty First-Step Proposal

Purpose: produce a supervisor-friendly first step that does not commit to complete vertical GAA fabrication.

## Recommended First Step

Prioritize literature-backed sheet / tube / wire comparison, equipment dependency extraction, process-risk mapping, and a simple baseline validation path.

## Preferred Near-Term Route

1. Build a vertical architecture comparison matrix for nanosheet, nanotube, and nanowire variants.
2. Extract equipment requirements and failure modes from current sources.
3. Identify the route with the shortest path to measurable electrical or photoelectric data.
4. Propose a minimal validation package, such as Ge/Si nanowire baseline I-V or photoresponse testing, before any full gate-stack commitment.

## Defer By Default

- Complete vertical GAA fabrication.
- Complete ALD-wrapped gate-stack integration.
- GeSn/Si vertical array fabrication.
- Cleanroom-intensive single-device processing without prior feasibility evidence.
